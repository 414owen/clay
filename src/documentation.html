Documentation

clay is an edsl that uses a monad to collect style rules

style rules are combinations of selectors combined with a list of properties

style rules be composed and be nested to create a stylesheet

stylesheets can be rendered to plain old css.

Types

to get start with clay it is useful to have several concepts clear

the most important types to know are Css, Selector, Key and Value, together with the Val type class

css is the monadic context in which we collect style rules and properties

rules map selectors to list of proprties, optinally combined with nested rules
